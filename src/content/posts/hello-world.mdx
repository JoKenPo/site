Bem vindo ao meu novo recanto na internet.

Esse espaço foi reiniciado com o objetivo de focar na legibilidade clássica da web ("brutalismo") e em máxima eficiência para ser hospedado via NGINX em um Raspberry Pi. Inspirado pela fantástica [Xe Iaso](https://xeiaso.net/).

## Tecnologias Escolhidas

Para não abandonar o ecossistema que uso profissionalmente, decidi usar o **Astro**. Além de suportar `.mdx`, ele consegue compilar para HTML puro, gerando 0 KB de JavaScript no cliente a menos que explícito. 

```javascript
// Exemplo de componente React que poderia ser inserido via MDX futuramente
import React from 'react';

export default function Counter() {
  const [count, setCount] = React.useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
```

> **Nota:** Usando CSS puro garantimos que o navegador perca pouquíssimo tempo interpretando estilos gigantes. É só cor e tipografia em sua forma mais pura e robusta.

## O que foi feito hoje
- Removemos fontes customizadas pesadas para focar 100% em leitura brutalista, economizando megabytes de banda.
- O site agora gera um [Feed RSS](/rss.xml) 100% funcional em tempo de compilação graças às Collections do Astro.
- Dockerizamos a versão final com a imagem levinha do NGINX Alpine e pareamos com a imagem do **Cloudflare Tunnels**.
- E o mais legal: subimos o blog **localmente em um Raspberry Pi 3** através de um SSD externo. Graças à engine _Zero Trust_ da Cloudflare, o tráfego do meu domínio bate seguro na porta da minha casa sem eu precisar abrir NENHUMA porta do roteador para a internet aberta!
