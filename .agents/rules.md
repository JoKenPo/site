# Diretrizes do Projeto

Este arquivo contém regras de comportamento para agentes de IA que trabalham neste repositório.

## Versionamento (Git)

- **Nunca comitar sozinho**: O agente não deve realizar `git commit` ou `git push` automaticamente por vontade própria. 
- **Aprovação explícita**: Commits só devem ser realizados após o usuário solicitar ou aprovar explicitamente a ação.
- **Padrão**: Seguir sempre o padrão de `conventional-commits` quando solicitado a comitar.
